# Donet World Control Panel (WCP)

A cluster internal client with a graphical user interface for interacting
with Distributed Objects at runtime, or updating fields of Distributed Objects
that are in the DBSS. If wanting to update a Distributed Object's `db` fields,
this tool removes the need for the developer to interact with the backend
database instance directly. Useful for debugging purposes.
