# DCFile Syntax Highlighting for VSCode

Support for the DistributedClass language, also known as the DC file.

Learn more at https://docs.donet-server.org/master/dclanguage/.

> **NOTE:** The author of this extension has written extensive documentation on the DistributedClass language, and [Panda3D](https://panda3d.org)'s Distributed Networking, under their own hobby project, [Donet](https://donet-server.org), that aims to implement the server technology behind [Panda3D](https://panda3d.org)'s Distributed Networking. **If you are looking for a mature implementation of the server, please visit https://astron.github.io/astron.**

The source code for this Visual Studio Code extension is available at https://gitlab.com/donet-server/donet/-/tree/master/tools/dcfile-syntax-highlighting.

## Features

Offers **basic syntax highlighting** via a TextMate grammar bundle that is loaded by Visual Studio Code.

![Syntax Highlighting Example](https://gitlab.com/donet-server/donet/-/raw/master/tools/dcfile-syntax-highlighting/example.jpg)

> **NOTE:** This extension does **not** (yet) provide a language server, therefore no fancy real-time error diagnostics, reference lookup, or auto complete.
