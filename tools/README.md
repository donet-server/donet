# Tools for Donet Developers

- `dcfile-syntax-highlighting` -> A VSCode extension that wraps the DCFile TextMate bundle for use within Visual Studio Code.
- `DCFile.tmbundle` -> A TextMate bundle for the DistributedClass (DCFile) grammar. Can be directly imported in JetBrains IntelliJ IDEs for basic syntax highlighting!
- `wireshark` -> Contains a plugin for Wireshark, written in Lua, that helps dissect network messages that use the Donet protocol.
